Why gngdb?

My initials are G.D.B.G., so I was going to have the github username gdbg. But, Fraser pointed out that people would think I was something to do with GNU Debugger, which I'm not. Luckily GNU already had the solution for this, so gngdb stands for:

  • Gngdb's not GNU Debugger
  • or
  • Gavin (n) Gray Douglas Buchanan

The first being a nested recursive acronym. Even if the second is a bit annoying, with the extra n.
